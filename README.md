# 📱 Projeto Redes Sociais

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

Esse é um projeto de front-end feito para simular uma tela de celular com atalhos para redes sociais. A ideia foi criar uma interface visual e interativa usando HTML, CSS, imagens e `iframe`.

## ✨ Sobre o projeto

A página principal mostra um celular no centro da tela e alguns botões laterais com ícones de redes sociais. Ao clicar em cada ícone, a tela dentro do celular muda, exibindo uma prévia da rede escolhida.

Em cada tela, o usuário deve rolar com o mouse ou com o touchscreen para chegar até o botão **ACESSE**, que leva para o link externo correspondente.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- `iframe`
- Imagens de fundo
- Links internos e externos
- Efeitos com `hover`
- Posicionamento com CSS

## 📚 O que eu pratiquei

- Criação de uma interface simulando um celular
- Uso de `iframe` para carregar páginas dentro de outra página
- Navegação entre páginas usando links com `target`
- Organização de arquivos em pastas
- Uso de imagens como fundo, ícones e telas
- Estilização de botões e efeitos ao passar o mouse
- Controle visual da rolagem nas páginas internas

## 💡 Tipo de projeto

Esse é um projeto de estudo focado em HTML e CSS. Ele ajudou a praticar a criação de layouts mais visuais, uso de imagens na composição da página e navegação entre conteúdos dentro de um mesmo espaço.
